# Utility Scripts

Miscellaneous utility scripts for data processing and helper functions.

## Available Scripts

### `simple_consolidate_banks.py`
Consolidates bank template data.

**Usage:**
```bash
python scripts/utils/simple_consolidate_banks.py
```

## Adding New Utilities

When adding utility scripts:
1. Place them in this directory
2. Add clear documentation at the top of the script
3. Update this README with usage instructions
4. Follow the project's coding standards

## Best Practices

- Keep utilities focused and single-purpose
- Add error handling and validation
- Include usage examples in docstrings
- Test thoroughly before committing
