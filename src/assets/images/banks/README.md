# Bank Logos Directory

This directory contains bank logos for the valuation application.

## Structure

Place bank logos with the following naming convention:
- File format: PNG or JPG (PNG recommended)
- File naming: `{BANK_CODE}.png` (lowercase)
- Image size: Recommended 120x120 pixels or similar square aspect ratio

## Examples

For the banks in your system, place logos as:
- `sbi.png` - State Bank of India
- `hdfc.png` - HDFC Bank
- `icici.png` - ICICI Bank
- `axis.png` - Axis Bank
- `pnb.png` - Punjab National Bank
- `bob.png` - Bank of Baroda
- `canara.png` - Canara Bank
- `ubi.png` - Union Bank of India
- `iob.png` - Indian Overseas Bank
- `uco.png` - UCO Bank
- `cbi.png` - Central Bank of India
- `boi.png` - Bank of India

## Fallback

If a bank logo is not found, the system will automatically use `default-bank.svg` as a fallback.

## Logo Guidelines

- Use official bank logos when possible
- Ensure logos are high quality and clear
- Maintain consistent sizing across all logos
- Use transparent backgrounds (PNG format recommended)
